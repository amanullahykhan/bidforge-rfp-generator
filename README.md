## ✨ Why BidForge AI?

**BidForge AI** is a lightweight, Python-based GUI application designed for teams that need full control over their Request for Proposal (RFP) response generation—without being locked into expensive SaaS platforms.

Built on a **pluggable AI architecture**, BidForge AI lets you swap between leading LLM providers (OpenAI, Anthropic, Gemini, DeepSeek) or run entirely locally via Ollama, all from a unified interface.

| | |
| :--- | :--- |
| 🔓 **Multi-Model Support** | Connect to OpenAI, Anthropic, Gemini, DeepSeek, and Local Ollama out of the box. |
| 🧠 **Knowledge Bank** | Inject your own company context using `.txt` or `.docx` files for accurate outputs. |
| 🗄️ **Local History DB** | Automatically saves your generation history and context locally using SQLite. |
| 💻 **Modern GUI** | Built with `customtkinter` for a clean, intuitive, dark-mode user interface. |
| 📄 **Native Export** | Easily export your generated drafts directly to Microsoft Word (`.docx`) format. |

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/amanulllahykhan/bidforge-ai.git](https://github.com/amanulllahykhan/bidforge-ai.git)
   cd bidforge-ai

2. **Install dependencies:**
Ensure you have Python installed, then run:
```bash
pip install customtkinter requests python-docx

```


3. **Run the application:**
```bash
python app.py

```



---

## 🛠️ Usage

1. Launch the app and enter your **API Key** or **Local Base URL** in the sidebar.
2. Navigate to the **Knowledge Bank** tab to drop in your context documents.
3. Switch to the **RFP Generator** tab, select your preferred tone (e.g., Executive, Technical), and paste the RFP requirements.
4. Click **Draft Response** and wait for the AI to generate the customized output.
5. Review, copy, or export the response in the **Output Review** tab.
